# Python Learning Sandbox

Welcome to my personal Python exploration repository. This project was created as a hands-on environment to master Python programming, understand its ecosystem, and write clean, structured code. 

Rather than relying solely on theoretical study, I built functional mini-projects and implemented core computer science concepts from scratch to ensure a deep, practical understanding of the language.

## Project Overview

This repository is divided into three main analytical areas:

* **Games:** A collection of interactive, logic-based mini-games built to practice control flows, user input handling, and game loop architecture.
* **Geometric Animations:** Mathematical rendering and terminal-based animations of geometric shapes, focusing on 3D rotations using projection matrices and trigonometry.
* **Data Structures:** Custom implementations of fundamental computer science data structures (such as Linked Lists, Nodes, and Edges). This section is dedicated to understanding memory management, Object-Oriented Programming (OOP) paradigms, and the underlying mechanics of iterators.

## Tech Stack & Tooling

* **Language:** Python 3
* **Package Manager:** [uv](https://github.com/astral-sh/uv) - A fast, modern Python package and project manager written in Rust.

I deliberately selected `uv` to manage this project to align with the current, cutting-edge standards of Python environment management and dependency resolution used in professional environments.
